# Fortune Telling App

タロットカードと星占いを提供する占いアプリケーション。

## 機能

- タロットカード占い
  - アニメーション付きのカードドロー
  - 詳細な解説付き
  - 1日1回の無料占い

- 星占い
  - 12星座対応
  - 対話形式の占い結果
  - 複数の運勢カテゴリー（恋愛、仕事、健康など）

## 技術スタック

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## 開始方法

```bash
# リポジトリのクローン
git clone https://github.com/t012093/fortune-telling-app.git

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## 環境変数

`.env`ファイルを作成し、必要な環境変数を設定してください：

```env
VITE_GEMINI_API_KEY=your_api_key_here
```
